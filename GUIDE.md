# 📝 CMEMO 사용법 가이드

CMEMO를 사용해 주셔서 감사합니다! 간단한 사용법을 안내해 드립니다.

### 🖱️ 기본 조작
* **제목 부근 드래그**: 메모 이동
* **우측 하단 드래그**: 메모 크기 조절
* **+ 버튼**: 새로운 메모 생성
* **톱니바퀴 버튼**: 개별 메모 설정 및 전역 설정
* **닫기(X) 버튼**: 메모 삭제 (마지막 메모 삭제 시 프로그램 종료)

### ⌨️ 단축키
* **Ctrl + Alt + Page Up**: 모든 메모를 맨 앞으로 가져오기 (바탕화면에서 가려졌을 때)
* **Ctrl + Alt + Page Down**: 모든 메모 숨기기

### ⚙️ 설정 메뉴
* **배경색**: 다양한 파스텔 톤 색상 선택
* **글꼴 및 크기**: 시스템 설치 글꼴로 변경 가능
* **테마**: 윈도우98 스타일, macOS 스타일 등 테마 적용
* **저장 위치**: 데이터 저장 경로(`.json`) 변경 및 백업 가능

### 💾 데이터 관리
* 모든 메모는 실시간으로 자동 저장됩니다.
* `정기 백업 설정`을 통해 데이터를 안전하게 관리하세요.

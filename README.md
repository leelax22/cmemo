# 📝 플로팅 메모 사용 가이드

본 프로그램은 가볍고 직관적인 데스크탑용 메모장입니다. 모든 데이터는 로컬 환경에 안전하게 저장됩니다.

### 💾 데이터 관리 및 백업

데이터 보호를 위해 제공되는 강력한 기능들을 활용해 보세요.

1. **자동 저장**: 모든 변경 사항(내용, 위치, 테마 등)은 즉시 자동 저장됩니다.
2. **📤 현재 데이터 백업**: 현재 작성된 모든 메모 상태를 스냅샷으로 남깁니다. `memo_storage_YYYYMMDD.json` 형태로 원하는 위치에 안전하게 보관할 수 있습니다.
3. **📁 백업 파일 불러오기**: 과거에 백업해둔 파일을 선택하여 현재 메모로 복구합니다.
4. **📅 정기 백업 설정**: 
   - 사용자가 설정한 주기(Cron 방식)에 따라 자동으로 백업을 수행합니다.
   - **자동 회전(Rotation)**: 설정한 개수만큼만 백업 파일을 유지하고 오래된 파일은 자동으로 삭제하여 용량을 효율적으로 관리합니다.
5. **💾 저장 위치 변경**: 기본 저장 경로를 USB나 클라우드 동기화 폴더(Google Drive 등)로 변경하여 사용할 수 있습니다.

### 🖱️ 기본 조작법

*   **메모 이동**: 제목 바(상단 영역)를 잡고 드래그하세요.
*   **크기 조절**: 모서리나 테두리를 드래그하여 조절 가능합니다.
*   **제목 수정**: 제목 영역의 ✏️ 아이콘을 클릭하여 수정할 수 있습니다.
*   **접기/펴기**: 제목 바를 **더블 클릭**하면 제목만 남고 접힙니다. 접힌 상태에 마우스를 올리면 내용이 일시적으로 보여 편리합니다.
*   **최상단 고정**: 📌 버튼을 누르면 다른 창에 가려지지 않는 '항상 위' 모드가 활성화됩니다.

### 🎨 스타일 상세 설정 (≡ 메뉴 버튼)

메모의 **설정(≡)** 버튼을 통해 나만의 스타일을 세밀하게 조절할 수 있습니다.

*   **🎨 배경색 선택**: 메모의 배경을 8가지 파스텔 톤으로 변경합니다.
*   **🔠 글꼴 변경**: 시스템에 설치된 폰트를 선택하여 전체 메모에 적용합니다.
*   **📏 제목 폰트 크기**: 메모 상단 제목의 크기를 별도로 조절할 수 있습니다. (9pt ~ 20pt)
*   **🔠 제목 폰트 Bold 설정**: 제목을 굵게 표시할지 여부를 선택합니다.
*   **📏 내용 폰트 크기**: 메모 본문의 글꼴 크기를 조절합니다. (9pt ~ 24pt)
*   **🖼️ 테마 변경**: 기본형, 둥근형, 헤더분리형 등 4가지 디자인 테마를 제공합니다.

### ⌨️ 단축키 및 트레이

*   **모든 메모 보기**: `Ctrl + Alt + Page Up`
*   **모든 메모 숨기기**: `Ctrl + Alt + Page Down`
*   **시스템 트레이**: 아이콘 우클릭으로 새 메모 생성, 가이드 보기, 백업 설정을 즉시 실행할 수 있습니다.
*   **트레이 더블 클릭**: 숨겨진 모든 메모를 즉시 화면 앞으로 가져옵니다.
